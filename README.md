# water jet
北京航空航天大学 2024 春季软件工程嵌入式项目

## 部署
运行部署脚本
1. 安装 pnpm
2. 安装 ros 依赖
3. 部署到 `/home/robot/catkin_ws` 下 
```bash
source deploy.sh
```
